# Jogo de adivinhação feito em Python na disciplina PCAP (Aula 16).

Nesse jogo o objetivo é adivinhar o número secreto selecionado pelo computador, podendo escolher as dificuldades 1- Fácil (1 a 10, 3 chances); 2 - Médio (1 a 100, 5 chances); 3 - Impossível (1 a 1000, 10 chances).

# Como jogar:

1. Abra o terminal na pasta aula16, e, em seguida, abra a pasta adivinhe.py;
2. Inicie o código;
3. Escolha um dos níveis citados acima, digitando 1, 2 ou 3 e tente adivinhar.

# O que eu pratiquei:

Variáveis, sub-rotinas, entrada, saída, decisão, repetição, operadores, etc.
